# CRUD-Application

# Como iniciar?
**(Necessário ter Node.js)**
1. Faça o download do projeto ou git clone
2. Abra o projeto aberto na IDE
3. Abra o terminal e digite: npm i ```(Para baixar dependências)```
4. Execute o projeto digitando: npm start ```(Para executar o projeto)```

## Objetivo
O projeto tem como intuito o estudo de vários conceitos que podem ser utilizados em projetos que englobam sistemas web, tal como:
- Banco de dados (SQLite)
- Autorização e tokens (JWT)
- Processamento server-side usando microframework (express)
- Tokens em cookies
Incluindo:
  - Routes
  - Controllers
  - Middleware
- Variáveis de ambientes
- hash de senhas (bycript)
- Componentização de elementos HTML

**Alguns trechos de códigos prematuros tiveram auxílio de IA, mas foram revisados na implementação final**
